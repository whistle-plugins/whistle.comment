# whistle.comment
可以给请求添加注释的插件，用法参考：https://www.hxstrive.com/subject/fiddler/1660.htm

# 安装
1. 安装最新版的 Whistle：
   ``` sh
    npm i -g whistle
    w2 restart
   ```
2. 安装插件：
    ``` sh
    w2 i whistle.comment
    ```
# 使用
通过右键菜单 `Others / Set Comment` 添加注释：
> 对方也要安装此插件才能看到注释
